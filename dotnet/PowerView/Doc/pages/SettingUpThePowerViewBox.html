Setting Up The PowerView Box

Describes preparing the computer which will run the PowerView application.

Linux

Generally the PowerView application requires mono 5.18.0 or greater if not otherwise specified.

Raspberry Pi

Before first boot

The recommended operating system is Raspberry PI OS. It can be installed easily with the Raspberry Pi Imager tool. Alternatively it can be installed with the NOOBS installer. Download the NOOBS zip available here. Once downloaded deploy it to the Raspberry Pi SD-card according to procedures, e.g. here.

Internet access is required on the RPi to download and install prerequisite software. Therefore be sure to connect an RJ45 patch cable to the gateway/router before first boot.

When booting the RPi the first time and NOOBS presents the available OS choices make sure to select the Raspberry PI OS option. Once installed and Raspbian boots configure Raspbian basics using the Raspberry Pi Configuration:

It is strongly recommended to change the pi password to something non-default. Also enable SSH (if a remote command line to the Raspberry Pi is required).

After first boot

When the RPi has been booted install prerequisite software using these command lines (note: They can take rather long to complete):

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install mono-complete

To verify the mono version execute this command line:

mono --version

Configuring network

It is strongly recommended to use ethernet cabling with the RPi, it is by far the most stable and easiest to maintain. "All" that is needed is an RJ45 patch cable connecting the RPi with the available gateway/router.

Setting up a wifi network is doable, however it takes one on a journey of well/ill supported USB wifi dongles, quality/crap RPi power supplys and configuring network files on the RPi, to name a few. Two hints:

  1. Get a quality RPi power supply, at least 1 A and proper voltages.
  2. WPA wifi security can be configured via /etc/network/interfaces file.

Either add the following to the /etc/network/interfaces file:

auto wlan0
allow-hotplug wlan0
iface wlan0 inet dhcp
  wpa-ssid "YourSsidInClearText"
  wpa-psk "YourWiFiWpaPasswordInClearText"

Alternatively if /etc/network/interfaces already has a wlan0 section looking like:

allow-hotplug wlan0
iface wlan0 inet manual
    wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf

Then add the following to the /etc/wpa_supplicant/wpa_supplicant.conf file:

network={
  ssid="YourSsidInClearText"
  scan_ssid=1
  key_mgmt=WPA-PSK
  psk="YourWiFiWpaPasswordInClearText"
}

Ask google for more assistance on how to setup wifi on the RPi.

Ubuntu

As for the Raspberry Pi install the mono using:

sudo apt-get install mono-complete

Windows

The PowerView application requires .NET Framework 4.7.2 or greater. If using a USB to RS232/RS422 adapter be sure to install the latest drivers available for the device.
