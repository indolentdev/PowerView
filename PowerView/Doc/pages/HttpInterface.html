HTTP Interface

PowerView provides a HTTP interface which supports JSON content.

Metering Value Ingestion

Endpoint for ingesting metering values into PowerView for storage and presentation.

Example Request:

POST /api/devices/livereadings HTTP/1.1
Host: [IP-Address-Or-HostName]:47362
Content-Type: application/json

Content Property Description:

Property Name Description
Items Array of meter reading objects
Label The user defined name of the metering performed. The label value should remain unchanged if the physical meter is replaced.
SerialNumber The serial number or id of the meter from which the register values originates. This value should change whenever the meter is exchanged.
Timestamp The universal coordinated time of the register values. Applies format: YYYY-MM-DDTHH-MM-SSZ
RegisterValues Array of register values read of the meter
ObisCode The DLMS/COSEM ObisCode identifying the value measured
Value The numeric value of the register. Consists only of digits, no thousand seperators or decimal points.
Scale The numer value which is the power of 10 multiplied with Value. Set to 0 to apply Value without multiplier.
Unit The unit of the value represent by Value and Scale.
