Installing PowerView

Describes installing and running PowerView.

Raspberry Pi

Install

Follow this procedure to install PowerView on a Raspberry Pi running Raspbian Stretch. It is assumed that the PowerView.exe.config file in the unzipped folder already has the correct configuration settings.

  1. Copy the unzipped folder containing PowerView to the Raspberry pi, e.g. using a USB flash dongle or via LAN. For this procedure /home/pi/PowerView is used for the temporary folder from where PowerView is installed.
  2. Start a terminal and change directory into /home/pi/PowerView
  3. Execute the command: chmod +x InstallPowerView.sh && sudo ./InstallPowerView.sh
  4. Assuming all went well remove the /home/pi/PowerView folder
  5. Reboot the box to launch PowerView as part of startup.

After install PowerView is placed in: /opt/PowerView/. PowerView process log file is: /var/log/PowerView.log.

Monitor log file

To view the PowerView log file one option is to use tail. In a terminal write: tail -n 100 -f /var/log/PowerView.log to follow the log file (do Ctrl+X Ctrl+C to exit the tail).

Stopping

PowerView can be stopped using the provided stop script: StopPowerView.sh. Change directory into /opt/PowerView and run: sudo ./StopPowerView.sh.

Linux

Depending on Linux distribution the Raspberry Pi procedure may work (should work on ubuntu). If not, then manually copy the PowerView folder to where you want it. The file named "PowerView" should be compatible with init.d requirements, though you may need to tailor some paths/files. If all else fails run PowerView using the command: sudo mono PowerView.exe.

Windows

The PowerView distribution package does not come with an installer for Windows, you are on your own. A rough guide; copy the PowerView folder to where you want it. To run PowerView simply run: PowerView.exe.

Visualizing readings

When PowerView is running and using a browser from a device connected to the home area network navigate to the following URL: http://{IP address of PowerView Box}:47362 which takes you to the default visualization page.
